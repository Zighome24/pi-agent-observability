Plan: Expand Steelman Agent's Visual Vocabulary

feature · medium

Task Description

We want to expand the Investment Steelman agent's visual vocabulary with a family of three new high-impact artifact types to make the bear case land faster and harder. Text-only descriptions of complex, spatial, or numerical timelines and ranges fail to command immediate comprehension. To solve this, we will build:

  1. A quote artifact: A typography-forward callout in the left pane rendering the exact "receipt" (CEO comments, regulator findings, short-seller lines) with a giant opening quote mark, italic style, and metadata (speaker, role, date, source link), plus room for bear commentary.
  2. A catalyst-timeline artifact: A left-to-right chronological timeline plotting key dated events (earnings miss → covenant breach → downgrade), color-coded by severity (bull/neutral/bear) to show domino pathways and pacing clearly.
  3. A valuation-gauge artifact: A horizontal range bar marking bear/base/bull target cases and placing a "where it trades today" indicator to show how much downside is already priced in.

Objective

Design and implement the three new visual artifacts across the entire Steelman codebase. Specifically, we will extend the agent extension types and prompt instructions, adjust the API database validation engine on the Bun server, and implement beautiful, dark-theme visual components in the Vue.js SPA frontend using styled HTML, responsive flexbox layout, and custom SVGs.

Problem Statement

Currently, the Steelman agent is limited to standard charts (bar, pie, trend), scorecards, risk-maps, raw text, and simple tables. When conveying high-impact evidence such as regulatory warnings or earnings calls quotes, sequential events, or range-bound stock valuations, the agent has to resort to plain text markdown or general tables. These standard formats lack the thematic pacing, typography weight, and spatial grouping necessary to immediately convince users of the bear argument's structure and severity.

Solution Approach

To implement this end-to-end, we will touch each layer of the Steelman stack:

  • Layer 1: Type Definitions & Schemas: Add "quote" | "catalyst-timeline" | "valuation-gauge" to the ArtifactKind Union in the extension, the web frontend, and the Bun API server.
  • Layer 2: Frontend Render Blocks: Write custom Vue rendering nodes inside App.vue's conditional rendering tree (using v-else-if) that parse the incoming selectedArtifact.data payloads. Make them responsive and dark-theme compliant.
  • Layer 3: System Prompting: Update STEELMAN_AGENT_SYSTEM_PROMPT.md to explicitly specify the JSON data structures and instruct the agent when and how to select these new types.

Relevant Files

  • apps/steelman/extension/steelman-product.ts: Pi Extension where ArtifactKind is registered for tool validation.
  • apps/steelman/extension/STEELMAN_AGENT_SYSTEM_PROMPT.md: The instructions loaded by the LLM agent to understand how to craft artifacts.
  • apps/steelman/web/src/App.vue: Vue frontend containing the tabs, sidebar state, and conditional artifact rendering.
  • apps/steelman/web/src/style.css: Global stylesheet containing colors, grid tokens, and styling blocks.
  • apps/steelman/server/src/server.ts: Backend router validating data models and managing runs.

Step by Step Tasks

  1. Update Core Types and Validation
    • Edit apps/steelman/extension/steelman-product.ts and update the ArtifactKind variable to include "quote", "catalyst-timeline", and "valuation-gauge".
    • Edit apps/steelman/server/src/server.ts and update type ArtifactKind to include the three new options.
    • Edit apps/steelman/web/src/App.vue and update type ArtifactKind.
  2. Create Core CSS Classes in style.css
    • Define styling rules for the typography-forward Quote (including custom italic rendering, metadata layout, and blockquote left-border accents).
    • Define styling rules for the left-to-right Catalyst Timeline (flex containers, severity colors, connector lines, and hover states).
    • Define styling rules for the Valuation Gauge range bar (custom horizontal slider tracks, gradient overlays, target markers, and animated pointers).
  3. Write Render Templates in App.vue
    • Add v-else-if="selectedArtifact.kind === 'quote'" render node with quotation icon, attribution link, and thesis commentary.
    • Add v-else-if="selectedArtifact.kind === 'catalyst-timeline'" render node displaying chronological events with color-coding.
    • Add v-else-if="selectedArtifact.kind === 'valuation-gauge'" render node representing valuation targets and current price point.
  4. Inject System Prompt Guidelines
    • Update STEELMAN_AGENT_SYSTEM_PROMPT.md with exact JSON shapes for the new types to ensure high-fidelity agent output.
  5. Smoke Tests & Validation Commands
    • Launch the development servers and verify the UI matches design specifications.

Testing Strategy

  • Unit & Schema Validation: Verify that emitting incorrect JSON payloads triggers valid TypeBox runtime errors, preventing database corruption.
  • UI Component Visual Verification: Test extreme bounds on components (e.g., long quotes, 8+ catalyst events, valuation figures with high ranges/negative valuations) to ensure responsive layout bounds hold.
  • Agent Prompt Alignment Test: Issue 5 target investment theses to the agent. Confirm that at least one Quote, one Timeline, and one Valuation Gauge are produced and cited as clickable `@refs` in the chat responses.

Acceptance Criteria

  • Quote Render: Large typographical quotation mark, bold/italic text body, metadata section with name, role, date, and clickable URL. Explicitly renders secondary commentary block without visual clipping.
  • Catalyst Timeline: Minimum of 3 nodes displayed horizontally. Nodes are clearly connected from left-to-right and display correct color code according to severity levels: bull (green), neutral (yellow), bear (red).
  • Valuation Gauge: Multi-value price target slider tracking "Bear" (left), "Base" (mid), and "Bull" (right). Displays "Current Price" indicator cleanly over the target range to show where it currently trades.
